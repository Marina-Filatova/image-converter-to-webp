# Image to WebP Converter

Конвертирует все JPG и PNG изображения в указанной папке в формат WebP.

## Возможности
- Пакетная конвертация JPG/PNG → WebP
- Сохранение прозрачности для PNG
- Настраиваемое качество (0–100)
- Пропуск уже существующих WebP-файлов
- Подробный вывод статистики

## Использование

```bash
# Базовая конвертация
python towebp.py /путь/к/папке

# С указанием качества
python towebp.py /путь/к/папке --quality 90

# Краткий вариант
python towebp.py /путь/к/папке -q 75
```
## Требования
* Python 3.7+
* Pillow (pip install Pillow)
